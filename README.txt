主板项目文件浏览器使用说明

功能特性
1. 文件树浏览：以树状结构展示项目文件组织
2. 智能文件夹筛选：自动识别以S/M开头的项目文件夹
3. 文件预览：支持多种文件格式的预览
   - 文本文件（.txt, .csv, .log, .bom, .drc, .rep, .rpt, .md, .json, .xml, .html, .htm, .ini, .cfg）
   - PDF文件（支持多页预览）
   - Excel文件（.xlsx, .xls）
   - Word文件（.docx, .doc）
   - 图片文件（.jpg, .jpeg, .png, .bmp, .gif, .tiff, .tif, .webp, .svg）
   - 视频文件（.mp4, .avi, .mov, .mkv, .flv, .wmv, .m4v, .webm, .mpg, .mpeg, .3gp）
   - 压缩包文件（.zip, .rar, .7z，树状结构展示）
   - Cadence OrCAD文件（.opj, .dsn, .sch, .brd, .bom, .dbk, .dsnlck）
4. 自定义项目结构生成：支持创建标准化的项目文件夹结构
5. 项目注释管理：支持为项目添加程序内注释
6. 搜索功能：支持按文件名搜索文件
7. 快捷键支持：F5刷新项目列表和文件树
8. 可配置项目路径：支持自定义项目文件来源路径
9. 设置持久化：自动保存程序配置和用户选择

安装步骤
1. 确保已安装Python 3.6或更高版本
2. 安装依赖库：
   pip install -r requirements.txt

运行程序
1. 在命令行中进入程序目录
2. 运行命令：
   python main.py

使用说明
1. 项目列表：程序自动显示D:\Desktop和D:\资料中以S/M开头的项目文件夹
2. 文件树浏览：点击项目列表中的项目，在右侧文件树中浏览文件和文件夹
3. 文件预览：点击文件树中的文件，在下方预览窗格查看文件内容
   - 文本文件：直接显示内容
   - 图片文件：显示图片预览，点击可查看大图
   - 视频文件：显示视频缩略图
   - 压缩包文件：以树状结构显示内部文件
   - Office文件：显示文件内容预览
4. 创建项目结构：点击"新建项目"按钮，选择版本和文件夹类型，生成标准化项目结构
5. 添加项目注释：右键点击项目，选择"添加注释"，输入程序内注释
6. 搜索文件：在搜索框中输入关键词，点击"搜索"按钮查找文件
7. 刷新内容：按F5键刷新项目列表和文件树
8. 配置项目路径：在设置中可以自定义项目文件来源路径

特殊文件处理
- 加密文件：.opj, .dsn, .sch, .brd, .bom, .dbk, .dsnlck等文件显示提示信息
- 大文件：自动限制显示内容，避免性能问题
- 缺失依赖：提供友好的安装提示
