# 主板项目文件浏览器 - 打包指南

## 系统要求
- Windows 10/11 操作系统
- Python 3.8 或更高版本
- 至少 2GB 可用内存

## 安装依赖

1. 确保已安装 Python 3.8+，并将 Python 添加到系统环境变量
2. 打开命令提示符或 PowerShell，导航到项目目录
3. 安装项目依赖：
   ```
   pip install -r requirements.txt
   ```
   主要依赖包括：
   - PyQt5：GUI框架
   - PyPDF2：PDF文件预览
   - openpyxl/xlrd：Excel文件预览
   - python-docx/olefile：Word文件预览
   - rarfile/py7zr：压缩包文件预览
   - opencv-python/numpy：视频缩略图生成
   - Pillow：图像处理
4. 安装打包工具 PyInstaller：
   ```
   pip install pyinstaller
   ```

## 执行打包

在项目目录下执行以下命令，使用预配置的spec文件：

```
pyinstaller main.spec
```

或使用完整命令：

```
pyinstaller -F -w --hidden-import="PyQt5.QtWidgets" --hidden-import="PyQt5.QtCore" --hidden-import="PyQt5.QtGui" --hidden-import="PyQt5.QtGui.QPixmap" --hidden-import="PyQt5.QtGui.QImage" --hidden-import="PyQt5.QtWidgets.QLabel" --hidden-import="PyQt5.QtWidgets.QScrollArea" --hidden-import="PyPDF2" --hidden-import="openpyxl" --hidden-import="docx" --hidden-import="xlrd" --hidden-import="olefile" --hidden-import="rarfile" --hidden-import="py7zr" --hidden-import="cv2" --hidden-import="numpy" --hidden-import="PIL" --hidden-import="subprocess" --hidden-import="traceback" -n "主板项目文件浏览器" main.py
```

参数说明：
- `-F`：生成单个可执行文件
- `-w`：隐藏控制台窗口（GUI应用程序需要）
- `--hidden-import`：指定需要包含的隐藏依赖
- `-n`：指定生成的可执行文件名称

## 查找生成的 EXE 文件

打包完成后，可执行文件将位于项目目录下的 `dist` 文件夹中，文件名为 `主板项目文件浏览器.exe`

## 运行程序

1. 直接双击 `主板项目文件浏览器.exe` 文件即可运行
2. 第一次运行时，程序会自动创建 `config.json` 配置文件

## 注意事项

1. 打包过程可能需要几分钟时间，请耐心等待
2. 生成的 EXE 文件较大（约 50-100 MB），这是因为包含了 Python 解释器和所有依赖库
3. 配置文件 `config.json` 会保存在与 EXE 文件相同的目录中
4. 如果打包过程中遇到错误，请确保：
   - 所有依赖库已正确安装
   - Python 版本符合要求
   - 当前用户有足够的权限创建文件和文件夹

## 配置文件保存说明

程序已优化配置文件路径处理，确保：

- 在开发环境中：配置文件保存在项目代码目录下
- 在打包后的 EXE 环境中：配置文件保存在 EXE 文件所在目录下

这样无论在什么环境下运行，程序都能正确加载和保存用户的设置。

## 手动创建配置文件

如果程序无法自动创建配置文件，可以手动创建 `config.json` 文件，内容如下：

```json
{
  "project_paths": [
    ["Desktop", "D:\\Desktop"],
    ["资料", "D:\\资料"]
  ]
}
```

## 常见问题解决方案

1. **程序启动后无响应**
   - 检查配置文件 `config.json` 是否存在且格式正确
   - 确保指定的项目文件夹路径存在

2. **提示缺少 DLL 文件**
   - 尝试重新安装 PyInstaller：`pip install --upgrade pyinstaller`
   - 或使用管理员权限运行打包命令

3. **打包后的程序无法预览某些文件**
   - 确保在打包前已正确安装所有依赖库
   - 特别是 PyPDF2、openpyxl 和 python-docx
